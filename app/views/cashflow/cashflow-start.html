As part of your application, you need to provide a detailed cashflow. This will help us calculate how much money you will need to receive each month.

During this stage of the application, you will need to provide us with information about:

  • the number of phases you are delivering this project in
  • how many homes you are building in each phase
  • the start date and completion date of each phase

  • construction method
  • square footage
  • projected sales value
